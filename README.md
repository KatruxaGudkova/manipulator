# 🦾 Четырёхзвенный манипулятор (Ant Colony Simulation)

Этот проект визуализирует **четырёхзвенный манипулятор**, использующий **алгоритм муравьиной колонии** для поиска оптимальной конфигурации с минимальным суммарным моментом звеньев.  
Отрисовка выполняется на HTML-canvas с динамическим построением манипуляторов в зависимости от введённых параметров.

---

## 🚀 Функциональность

- Ввод исходных данных через HTML-форму  
- Отрисовка координатной сетки и осей  
- Построение манипуляторов с разными конфигурациями  
- Поиск минимального суммарного момента в вершинах звеньев  
- Выделение оптимального манипулятора зелёным цветом  
- Визуализация и расчёты выполняются прямо в браузере  

---

## 🧩 Используемые технологии

- **HTML5 Canvas** — визуализация сетки и манипулятора  
- **Vanilla JavaScript** — реализация алгоритма и логики  
- **CSS3** — оформление интерфейса формы и канваса

## 🖥️ Как запустить проект
1. Склонируй репозиторий или просто скачай файлы
2. Открой файл index.html в браузере.
3. Введи параметры в форму:
    X₀, Y₀ - координаты начальной точки фиксации
    X₄, Y₄ - координаты конечной точки схвата
    L - длина одного звена
    M - масса одного звена
    N - количество манипуляторов (итераций)
    pᵤ - коэффициент отложения феромона
    qᵤ - коэффициент испарения феромона
4. Нажми кнопку «ПОСТРОИТЬ МАНИПУЛЯТОР»
5. На canvas появится изображение манипуляторов, где:
- зелёный цвет - оптимальный вариант с минимальным моментом
- серые линии - остальные варианты.

## 🧠 Ключевые функции
| Функция                   | Назначение                                         |
| ------------------------- | -------------------------------------------------- |
| `drawGrid()`              | Отрисовка сетки и координатных осей                |
| `drawLink()`              | Рисует звено между двумя точками                   |
| `drawOM()`                | Отображает основную ось манипулятора               |
| `intersectionOfCircles()` | Решает задачу пересечения двух окружностей         |
| `minMoment()`             | Вычисляет минимальный суммарный момент звеньев     |
| `antColony()`             | Основной алгоритм муравьиной колонии               |
| `findPhi()`               | Вычисляет угол между двумя точками                 |
| `randomInRange()`         | Возвращает случайное значение в заданном диапазоне |

## 📊 Алгоритм муравьиной колонии (Ant Colony)

Каждый "муравей" строит вариант манипулятора, выбирая углы в пределах диапазона.
На основе полученных моментов происходит перераспределение вероятностей (феромонов),
что постепенно приводит к поиску наиболее устойчивого положения с минимальным моментом.
